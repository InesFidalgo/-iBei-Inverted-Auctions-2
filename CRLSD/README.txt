MEETO

Collaboration and Social Networking

Instalação:
	Para uma mais fácil utilização do Meeto encontram-se na pasta bin, os seguintes ficheiros:
	RMIServer.sh
		Execute este script para correr o servidor RMI. Deve dar, como parâmetro, o ip da máquina que contém a base de dados a utilizar

	Meta2_war.war
		Este ficheiro deverá ser colocado na pasta webapps da sua instalação do Apache Tomcat, de forma a fazer deploy da mesma e poder aceder ao website. Deve ser acompanhado do ficheiro policy.all, também fornecido.

	Create_New_Tables.sql
		Este script pode ser usado para criar as tabelas usadas pelo programa, numa base de dados Oracle SQL

Utilização:
	Para usar o Meeto, deverá aceder ao website e registar um utilizador novo, ou fazer login se já tiver conta. Depois disso, tem acesso ao menu de opções, onde poderá ver os seus convites pendentes e as suas reuniões. Poderá também, caso pretenda, fazer logout. No menu de convites, aparecem os vários convites que lhe tiverem sido feitos para reuniões, bem como a opção de aceitar cada um deles. No menu de reuniões, poderá ver as informações gerais e as ordens de trabalho de cada uma delas, que poderá editar, e convidar outros utilizadores para uma reunião.
